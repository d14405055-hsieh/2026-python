# 三國武將 PK 遊戲

這是依據 week-07 HOMEWORK 規格延伸出的可視覺化版本。

## 檔案說明

- chibi_warlords_game.py：主程式（Tkinter 遊戲畫面）
- generals.txt：武將資料，EOF 結尾
- battles.txt：對戰陣容設定，EOF 結尾
- launch_game.bat：雙擊即可啟動遊戲
- create_desktop_shortcut.ps1：建立桌面捷徑
- make_shortcut.bat：一鍵呼叫 PowerShell 建立捷徑

## 遊玩方式

1. 雙擊 launch_game.bat
2. 你的隊伍是蜀吳聯軍，敵人是魏軍
3. 回合開始可先選擇目標，再決定動作：普通攻擊、必殺技、防禦、補給
4. 打倒全部魏軍即勝利

## 操作說明（按鍵與滑鼠）

- 滑鼠操作：
	- 點選「目標」下拉選單決定攻擊對象
	- 點選按鈕執行「普通攻擊 / 必殺技 / 防禦 / 補給」
- 鍵盤快捷鍵：
	- 1：普通攻擊
	- 2：必殺技
	- 3：防禦
	- 4：補給
	- Tab：切換下一個敵方目標
	- H：開啟規則說明
	- R：重新開局
	- M：切換模式（新手 / 老手 / AI自動）

## 雙人對戰操作

- 1號玩家（蜀吳聯軍）：
	- W：普通攻擊
	- A：必殺技
	- S：防禦
	- D：補給
- 2號玩家（曹魏軍）：
	- 上：普通攻擊
	- 左：必殺技
	- 下：防禦
	- 右：補給
- 雙人模式可用 M 切換到「雙人對戰」

註：沒有方向鍵移動，這是回合制戰鬥遊戲，重點是「選目標 + 選動作」。

## 新增內容

- 可動武將圖像：戰場上每名武將會持續待機動作
- 戰鬥特效：普通攻擊有斬擊與受擊閃光
- 必殺技專屬動畫：
	- 火系武將：火鳳燎原
	- 軍師型武將：八陣回春
	- 近戰武將：無雙斬
- 規則說明按鈕：可直接查看新手教學
- 戰場 UI 升級：左我方武將欄、右敵方武將欄、中央主戰場 HUD、底部技能面板
- 隊伍總血量條：中央戰場上方顯示雙方整體戰況

## 遊戲模式

- 新手模式：我方較強、補給次數較多，適合第一次遊玩
- 老手模式：敵方更強、AI 更兇，會有時反殺你，補給較少，節奏更快
- AI自動模式：由電腦控制我方與敵方，適合觀戰與學習流程
- 雙人對戰：本機熱座對戰，1P 用 WASD，2P 用方向鍵

## 建立桌面捷徑

1. 雙擊 make_shortcut.bat
2. 桌面會出現 Sanguo-Game.lnk
3. 之後可直接從桌面捷徑開啟遊戲

## 課程對應重點

- 檔案 I/O：讀取 generals.txt 與 battles.txt，EOF 停止
- namedtuple：General 結構
- sorted：依速度排回合
- Counter：統計輸出傷害
- defaultdict：紀錄承受傷害
